## Description: <br>
Video generation skill that helps an agent choose an appropriate dLazy CLI video model from the user's prompt. <br>

This skill is ready for commercial/non-commercial use. <br>

## Publisher: <br>
[dlazyai](https://clawhub.ai/user/dlazyai) <br>

### License/Terms of Use: <br>
MIT-0 <br>


## Use Case: <br>
Developers, creators, and agents use this skill to generate or transform videos with dLazy's hosted media-generation service through the pinned dLazy CLI. <br>

### Deployment Geography for Use: <br>
Global <br>

## Known Risks and Mitigations: <br>
Risk: The skill requires a dLazy API key and sends prompts or user-provided media to dLazy's cloud service. <br>
Mitigation: Use a revocable dLazy API key, review the dLazy CLI/source and service terms before installing, and only provide media that is acceptable to upload to dLazy. <br>
